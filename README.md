# 🏹 RobinHood P2P v1.2 stable

Защищённый децентрализованный мессенджер с аудиозвонками, голосовыми сообщениями, сквозным шифрованием и добровольной P2P-сетью.

Без регистраций и логинов. Только вы и собеседник. Пользователи помогают сети становясь пирами.

## 🔐 Безопасность

| Уровень | Технология |
|---------|------------|
| Шифрование сообщений | AES-256-GCM |
| Защита канала | tempKey → SHA-256, сервер знает только хеш |
| Верификация пира | peerId привязка маяка к создателю |
| Хранение ключей | IndexedDB |
| Физическая защита | PIN-код / графический ключ + самоуничтожение |
| Звонки | DTLS-SRTP через WebRTC |

## 🚀 Как начать

1. Откройте приложение
2. Нажмите 🏹 Стрела — создайте канал
3. Отправьте ссылку собеседнику
4. Дождитесь подключения — канал активируется
5. Переписывайтесь, отправляйте голосовые и звоните

## 🏗️ Технологии

| Компонент | Назначение |
|-----------|------------|
| P2PPong | Сигнальный сервер (Render + Workers) |
| AES-256-GCM | Сквозное шифрование |
| WebRTC + DTLS-SRTP | Аудиозвонки |
| IndexedDB | Хранение контактов и фона |
| PWA | Установка на телефон |
| Lottie | Анимации |
| D1 (Cloudflare) | Хранение списка пиров |
| Workers KV | Хранение маяков и сообщений |

## 🤝 Стать пиром

Включите «Стать пиром» — ваш узел появляется в D1-трекере. Другие могут найти вас по Peer ID. Трафик ~2-5 КБ/час. При выключении — узел удаляется из трекера.

## 📲 PWA

Установите через меню браузера → «Добавить на экран».

## ⚠️ Важно

- Ключи только у вас
- Сервер знает только хеш ключа
- Сверяйте Peer ID с собеседником
- Маяки живут 20 минут
