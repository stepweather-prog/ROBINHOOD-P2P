# 🏹 RobinHood P2P

**Защищённый децентрализованный мессенджер. Бета-версия.**

Никаких серверов, регистраций, логинов. Только вы и собеседник.

Сообщения шифруются сквозным ключом AES-256-GCM. Поддерживает текстовые сообщения, эмодзи, прямые голосовые звонки (P2P), темы оформления и туннели WireGuard.

---

## Как начать

1. Откройте приложение
2. Нажмите **Стрела** → выберите тип стрелы
3. Отправьте ссылку другу
4. Когда друг откроет ссылку — канал активируется
5. Можно переписываться и звонить!

---

## Стрелы

|     Стрела     |  Сервер  |    Скорость   | Где работает  |
|----------------|----------|---------------|---------------|
| 🏕️ **Шервуда** | Render   |   2-6 сек     | Везде         |

## Звонки

- Прямой P2P через WebRTC
- Шифрование DTLS-SRTP
- Авто-выбор STUN/TURN
- Громкая связь
- Мелодия входящего, гудки, звук соединения

---

## Технологии

|    Компонент       |     Назначение        |
|--------------------|-----------------------|
| **P2PPong**        | Сигнальный сервер     |
| **AES-256-GCM**    | Шифрование сообщений  |
| **WebRTC**         | Голосовые звонки      |
| **WireGuard**      | Туннели (опционально) |
| **PWA**            | Установка на телефон  |
