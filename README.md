# 🏹 RobinHood P2P v1.2 beta

Защищённый децентрализованный мессенджер с аудиозвонками, голосовыми сообщениями, видео-кружками и сквозным шифрованием.

Без обязательных серверов, регистраций и логинов. Только вы и собеседник. Пользователи могут добровольно помогать сети, становясь пирами.

## 🔐 Безопасность

| Уровень | Технология |
|---------|------------|
| Шифрование сообщений | AES-256-GCM |
| Защита канала | tempKey генерируется на клиенте, сервер знает только SHA-256 |
| Хранение ключей | IndexedDB |
| Физическая защита | PIN-код / графический ключ + самоуничтожение |
| Медиа | DTLS-SRTP для звонков |

## 🚀 Как начать

1. Откройте приложение
2. Нажмите 🏹 Стрела → создайте канал
3. Отправьте ссылку собеседнику
4. Дождитесь подключения — канал активируется
5. Переписывайтесь, отправляйте голосовые, видео-кружки и звоните

## 🏗️ Технологии

| Компонент | Назначение |
|-----------|------------|
| P2PPong | Сигнальный сервер (Render + Workers) |
| AES-256-GCM | Сквозное шифрование |
| WebRTC + DTLS-SRTP | Голосовые звонки |
| IndexedDB | Хранение контактов и фона |
| PWA | Установка на телефон |
| Lottie | Анимации |
| WebTorrent DHT | Добровольная P2P помощь сети |

## 📲 PWA

Приложение можно установить на телефон через меню браузера → «Добавить на экран».

## 🤝 Стать пиром

Включите опцию «Стать пиром» в настройках чтобы помогать другим пользователям находить друг друга. Трафик минимальный (~2-5 КБ/час), батарея не расходуется в фоне. Вы не пересылаете чужие сообщения — только сигнальные данные.

## ⚠️ Важно

- При обновлении страницы канал сохраняется
- Ключи шифрования хранятся только у вас
- Сервер не знает секретов каналов
- Сверяйте Peer ID с собеседником
