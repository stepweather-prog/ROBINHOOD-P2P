# RobinHood P2P & RobinCall

Полностью децентрализованные P2P-приложения: мессенджер и аудиозвонки без серверов.

## Как начать

1. Открой RobinHood P2P
2. Нажми «Открыть колчан» → «Крафт стрелы»
3. Отправь QR-код или Beacon ID другу
4. Сверьте 7-значный код голосом
5. Чат открыт. Для звонков — RobinCall, та же схема.
