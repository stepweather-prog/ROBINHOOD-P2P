# 🏹 RobinHood P2P

**Защищённый децентрализованный мессенджер с аудио/видеозвонками, голосовыми сообщениями и сквозным шифрованием.**
Версия 1.2

Без серверов, регистраций и логинов. Только вы и собеседник.
Сообщения шифруются AES-256-GCM. Поддерживает текстовые сообщения, эмодзи, голосовые сообщения, прямые P2P-звонки (аудио/видео), темы оформления, туннели WireGuard и самоуничтожающиеся сообщения.

---

## Как начать

1. Откройте приложение
2. Нажмите **🏹 Стрела** → выберите тип канала
3. Отправьте ссылку собеседнику
4. Когда собеседник откроет ссылку, канал активируется
5. Можно переписываться, отправлять голосовые сообщения и звонить

## Типы каналов (стрел)

|     Стрела     |  Сервер  |    Скорость   | Где работает  |

## Сообщения

- ✍️ Текстовые сообщения с индикатором набора
- 🎤 Голосовые сообщения (запись и воспроизведение прямо в чате)
- 📎 Отправка файлов (изображения, документы, аудио)
- 💣 Самоуничтожающиеся сообщения
- 👍 Реакции на сообщения (эмодзи)
- 🔗 Автоматическое определение ссылок и предпросмотр
- 📜 История чата с шифрованием (опционально)

---

## Звонки

- Прямой P2P через WebRTC (аудио и видео)
- Шифрование DTLS-SRTP
- Авто-выбор STUN/TURN
- Громкая связь, отключение микрофона/камеры
- Мелодия входящего, гудки, звук соединения
- Видеоконтейнеры (свой/собеседника)

---

## Интерфейс

- Закреплённый статус Robin Hood (не прокручивается)
- Адаптивная панель ввода с эмодзи и голосовыми сообщениями
- Окно приглашения с возможностью отмены и копирования ссылки
- Слайд-панели для контактов и настроек
- Анимации лучника при отправке

## Технологии

|    Компонент       |     Назначение        |
|--------------------|-----------------------|
| **P2PPong**        | Сигнальный сервер     |
| **AES-256-GCM**    | Шифрование сообщений  |
| **WebRTC**         | Голосовые и видеозвонки |
| **WireGuard**      | Туннели (опционально) |

## Важно

- Никто не может прочитать переписку — ключи только у вас
