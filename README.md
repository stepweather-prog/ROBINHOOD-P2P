# 🏹 RobinHood P2P v3.1

**Защищённый децентрализованный мессенджер** с аудио/видеозвонками, голосовыми сообщениями, видео-кружками и сквозным шифрованием.

Без серверов, регистраций и логинов. Только вы и собеседник.

---

## 🔐 Безопасность

| Уровень | Технология |
|:---|:---|
| Шифрование сообщений | AES-256-GCM + ECDH P-256 |
| Совершенная прямая секретность | Double Ratchet (каждое сообщение — новый ключ) |
| Верификация | 12-значный Safety Number (сверка голосом) |
| Защита канала | tempKey генерируется на клиенте, сервер знает только SHA-256 |
| Хранение ключей | IndexedDB + мастер-пароль (PBKDF2, 100 000 итераций) |
| Физическая защита | PIN-код / графический ключ + самоуничтожение |
| Медиа | DTLS-SRTP + Safety Number для звонков |
| Метаданные | Обфускация трафика + фейк-пакеты |

---

## 🚀 Как начать

1. Откройте [приложение](https://stepweather-prog.github.io/ROBINHOOD-P2P/)
2. Нажмите 🏹 **Колчан** → создайте канал
3. Отправьте ссылку собеседнику
4. Дождитесь подключения — канал активируется
5. Переписывайтесь, отправляйте голосовые и звоните

## 📋 Возможности

### Сообщения
- ✍️ Текстовые сообщения с индикатором набора
- 🎤 Голосовые сообщения (запись и воспроизведение в чате)
- 📹 Видео-кружки
- 📎 Отправка файлов (изображения, документы, аудио)
- 💣 Самоуничтожающиеся сообщения
- 👍 Реакции на сообщения
- 🔗 Автоматическое определение ссылок

### Звонки
- 📞 Прямой P2P через WebRTC (аудио и видео)
- 🔒 Шифрование DTLS-SRTP
- 🔊 Громкая связь, отключение микрофона/камеры
- 🎵 Мелодия входящего, гудки, звук соединения
- 📹 Видеоконтейнеры (свой/собеседника)

### Интерфейс
- 🎨 18 тем оформления (включая «Золотой Пергамент», «Ночной Костёр»)
- 🍂 Анимированные листья, искры при отправке
- 📱 Тактильная обратная связь
- 👆 Свайп-жесты для навигации
- 🔒 Экран блокировки (PIN / графический ключ)
- 🔐 Мастер-пароль для ключей шифрования

## 🏗️ Технологии

| Компонент | Назначение |
|:---|:---|
| P2PPong | Сигнальный сервер (Render + Workers) |
| AES-256-GCM + ECDH | Сквозное шифрование |
| Double Ratchet | Совершенная прямая секретность |
| WebRTC + DTLS-SRTP | Голосовые и видеозвонки |

| IndexedDB | Хранение ключей |
| PWA | Установка на телефон |
| Lottie | Анимации |

## 📲 PWA

Приложение можно установить на телефон через меню браузера → «Добавить на экран». В режиме PWA доступно до 20 каналов (в браузере — 2).

---

## ⚠️ Важно

- При обновлении страницы канал сохраняется (сессии шифруются)
- Ключи шифрования хранятся только у вас
- Сервер не знает секретов каналов
- Сверяйте Safety Number с собеседником голосом
