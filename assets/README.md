RobinHood P2P
Защищённый децентрализованный мессенджер. Бета-версия. Никаких серверов, регистраций, логинов. Только вы и собеседник.

Как начать
Откройте приложение. Сверху шапка с аватаром и ником. Нажмите на аватар — смените лучника. Нажмите на ник — измените имя (до 10 символов).

Нажмите Инвайт Стрела — приложение создаст защищённый канал, ссылка скопируется в буфер. Отправьте её другу.

Когда друг откроет ссылку — приложение сверит ключи, создаст мост. Вы оба увидите: «✅ Колчан открыт!» Всё, можно переписываться. Сообщения шифруются сквозным ключом.

Возможности
Аватар — смена своего лучника

Лучники — список контактов. Долгое нажатие: переименовать, сменить аватар друга, уволить лучника

Перестрелки — быстрый переход между чатами

Инвайт Стрела — создать новый канал (в браузере до 2, в приложении до 20)

Забыть — управление историей. Если выключено — секреты живут только до закрытия приложения

Звук — включение/выключение звука

Курить — очистка переписки

Важно
При обновлении страницы канал сбрасывается — отправьте новую Инвайт Стрелу

Для большего числа чатов установите приложение на телефон

Никто не может прочитать вашу переписку — ключи знают только ваши устройства
